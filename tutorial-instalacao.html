Instalação

Como instalar a extensão SummFlux.

Este tutorial orienta a instalação no Google Chrome para uso com WhatsApp Web. Ajuste os links de download quando a extensão estiver publicada ou quando você enviar o pacote para clientes.

Vídeo rápido

Instalação pela Chrome Web Store.

Pesquise por SummFlux, confirme a extensão com a primeira logo da empresa e clique em instalar.

1. Requisitos

Use um computador com Google Chrome atualizado, acesso ao WhatsApp Web e uma conta ativa na SummFlux. O usuário precisa estar autorizado pela empresa para utilizar a extensão e enviar análises ao CRM.

2. Instalação pela Chrome Web Store

O cliente deve acessar a Chrome Web Store, pesquisar por SummFlux, confirmar que a extensão exibida usa a primeira logo da empresa e clicar em Usar no Chrome. Após confirmar a instalação, o ícone da SummFlux ficará disponível na barra de extensões do navegador.

3. Instalação manual durante testes

1

Baixe o pacote

Receba o arquivo da extensão enviado oficialmente pela SummFlux e extraia a pasta em um local seguro.

2

Abra as extensões

No Chrome, acesse chrome://extensions e ative o Modo do desenvolvedor.

3

Carregue sem compactação

Clique em Carregar sem compactação e selecione a pasta da extensão.

4

Acesse o WhatsApp Web

Abra o WhatsApp Web, entre em uma conversa e utilize o botão da SummFlux para analisar e enviar ao CRM.

4. Permissões

A extensão precisa de permissões para funcionar dentro do WhatsApp Web e comunicar-se com o backend da SummFlux. Essas permissões servem para ler a conversa ativa quando o usuário solicita análise e enviar os dados ao CRM da empresa.

5. Uso correto

O vendedor deve utilizar a extensão apenas em conversas relacionadas à operação comercial da empresa. A análise deve ser confirmada antes de salvar o lead, evitando registros incorretos ou informações fora de contexto.

6. Problemas comuns

Se o botão não aparecer, recarregue o WhatsApp Web, verifique se a extensão está ativa, confirme se o usuário está autorizado e teste novamente. Se ainda assim não funcionar, envie print da tela e versão do Chrome para o suporte.
